# Shadcn Admin Template

## 소개
이 프로젝트는 Next.js, Shadcn 을 기반으로 한 Admin Template입니다.
TypeScript와 최신 웹 기술을 활용하여 구축되었습니다.

## 기술 스택
- **프레임워크**:
  - Next.js (App Router)
- **언어**:
  - TypeScript
- **UI 컴포넌트**:
  - Shadcn UI
- **스타일링**:
  - Tailwind CSS
- **상태 관리**: 
  - Zustand

## 시작하기

### 필요 조건
- Node.js v22
- Next.js v15
- React v19
- tailwindcss v4
- pnpm

### 설치 방법
```bash
# 저장소 클론
git clone https://github.com/harry-js/admin-template.git

# 의존성 설치
pnpm install

# 개발 서버 실행
pnpm dev
```
